## How to start (如何开始)

1. clone project to local
   
```bash
   git clone git@github.com:Frioa/cloud_music.git
 ```

2. install [Flutter](https://flutter.io/docs/get-started/install)


3. build & run
   
 ```bash
 flutter run apk
 ```
